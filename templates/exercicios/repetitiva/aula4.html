🎯 Objetivo

  • Controlar o fluxo dos loops com break e continue
  • Saber quando usar cada comando
  • Otimizar loops para melhor performance

🛑 O comando break

O break interrompe completamente o loop e sai dele imediatamente.

Exemplo Básico:

for i in range(10):
    if i == 5:
        break  # Para quando i for 5
    print(i)

# Saída: 0, 1, 2, 3, 4

⏭️ O comando continue

O continue pula a iteração atual e vai para a próxima.

Exemplo Básico:

for i in range(10):
    if i == 5:
        continue  # Pula quando i for 5
    print(i)

# Saída: 0, 1, 2, 3, 4, 6, 7, 8, 9

🔍 Exemplos Práticos com break

1. Busca em Lista

nomes = ["Ana", "Bruno", "Carlos", "Diana", "Eduardo"]
nome_procurado = "Carlos"

for nome in nomes:
    print(f"Verificando: {nome}")
    
    if nome == nome_procurado:
        print(f"✅ Encontrado: {nome}")
        break
else:
    print("❌ Nome não encontrado")

2. Validação com Limite de Tentativas

senha_correta = "python123"
max_tentativas = 3

for tentativa in range(1, max_tentativas + 1):
    senha = input(f"Tentativa {tentativa} - Digite a senha: ")
    
    if senha == senha_correta:
        print("✅ Acesso liberado!")
        break
    else:
        print("❌ Senha incorreta")
        
    if tentativa == max_tentativas:
        print("🔒 Acesso negado - muitas tentativas")

3. Menu com Saída

while True:
    print("\n=== CALCULADORA ===")
    print("1. Somar")
    print("2. Subtrair")
    print("3. Multiplicar")
    print("4. Sair")

⏭️ Exemplos Práticos com continue

1. Filtrando Números Pares

print("Números ímpares de 1 a 10:")

for i in range(1, 11):
    if i % 2 == 0:  # Se for par
        continue    # Pula para o próximo
    
    print(i)  # Só imprime ímpares

2. Processando Lista com Exceções

valores = ["10", "20", "abc", "30", "xyz", "40"]

soma = 0
for valor in valores:
    try:
        numero = int(valor)
    except ValueError:
        print(f"⚠️ Ignorando valor inválido: {valor}")
        continue  # Pula para o próximo
    
    soma += numero
    print(f"✅ Adicionado: {numero}")

print(f"Soma total: {soma}")

3. Validação de Dados

emails = ["user@email.com", "invalido", "admin@site.com", "erro.com"]

emails_validos = []

for email in emails:
    if "@" not in email or "." not in email:
        print(f"❌ Email inválido: {email}")
        continue
    
    emails_validos.append(email)
    print(f"✅ Email válido: {email}")

print(f"\nEmails válidos encontrados: {len(emails_validos)}")

🔄 Loops Aninhados com break/continue

Break em Loop Aninhado

# break só afeta o loop mais interno
for i in range(3):
    print(f"\nLinha {i}:")
    
    for j in range(5):
        if j == 3:
            break  # Só sai do loop j
        print(j, end=" ")

# Saída:
# Linha 0: 0 1 2
# Linha 1: 0 1 2  
# Linha 2: 0 1 2

Continue em Loop Aninhado

for i in range(3):
    for j in range(5):
        if j == 2:
            continue  # Pula j=2
        print(f"({i},{j})", end=" ")
    print()  # Nova linha

# Saída:
# (0,0) (0,1) (0,3) (0,4)
# (1,0) (1,1) (1,3) (1,4)
# (2,0) (2,1) (2,3) (2,4)

🏷️ Usando Labels com break (Avançado)

Python não tem labels como outras linguagens, mas você pode usar flags:

encontrou = False

for i in range(3):
    for j in range(3):
        if i == 1 and j == 1:
            print(f"Encontrado em ({i},{j})")
            encontrou = True
            break
    
    if encontrou:
        break  # Sai do loop externo também

print("Busca finalizada")

⚖️ Quando Usar break vs continue

✅ Use break quando:

  • Encontrou o que estava procurando
  • Atingiu uma condição de parada
  • Detectou um erro crítico
  • O usuário quer sair do programa

✅ Use continue quando:

  • Quer pular elementos inválidos
  • Filtrando dados de uma lista
  • Ignorando casos especiais
  • Validando entrada sem parar

🚀 Busca Eficiente com break

# Verificar se número é primo
numero = int(input("Digite um número: "))

if numero < 2:
    print(f"{numero} não é primo")
else:
    eh_primo = True
    
    # Testar divisores até a raiz quadrada
    for i in range(2, int(numero ** 0.5) + 1):
        if numero % i == 0:
            eh_primo = False
            print(f"Encontrou divisor: {i}")
            break  # Para no primeiro divisor!
    
    if eh_primo:
        print(f"✅ {numero} é primo!")
    else:
        print(f"❌ {numero} não é primo")

💡 Dicas Importantes

  • 🛑 break sai completamente do loop
  • ⏭️ continue pula apenas a iteração atual
  • 🎯 Use break para buscas eficientes
  • 🔍 Use continue para filtrar dados
  • 🔄 Em loops aninhados, afetam apenas o loop mais interno
  • ⚡ break pode melhorar muito a performance
