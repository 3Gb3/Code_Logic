🎯 Objetivo

  • Implementar sistema de autenticação simples
  • Validar senha até acerto
  • Dar feedback ao usuário

📚 Conceitos: Autenticação com Loop

Um padrão comum é pedir uma senha até o usuário acertar:

Estrutura básica:

senha_correta = "1234"
senha = input("Digite a senha: ")

while senha != senha_correta:
    print("Senha Invalida! Tente novamente.")
    senha = input("Digite a senha: ")

print("Acesso Permitido!")

💡 Variações:

  • Tentativas limitadas: Adicionar contador de tentativas
  • Case sensitive: Diferenciar maiúsculas/minúsculas
  • Mensagens personalizadas: Informar número de tentativas restantes

🔒 Conceitos de Segurança (Básico)

⚠️ NOTA IMPORTANTE: Este é apenas um exercício didático. Em aplicações reais:

❌ NÃO faça (como neste exercício):

  • Senha hard-coded (fixa no código)
  • Senha em texto plano
  • Tentativas ilimitadas

✅ Em produção, USE:

  • Hash de senhas (bcrypt, argon2)
  • Banco de dados seguro
  • Limite de tentativas
  • Autenticação de dois fatores
  • HTTPS/SSL

Exemplo melhorado (limite de tentativas):

senha_correta = "2002"
tentativas = 0
max_tentativas = 3

senha = input("Digite a senha: ")
tentativas += 1

while senha != senha_correta and tentativas < max_tentativas:
    print(f"Senha Invalida! Tentativa {tentativas}/{max_tentativas}")
    senha = input("Digite a senha: ")
    tentativas += 1

if senha == senha_correta:
    print("Acesso Permitido!")
else:
    print("Conta bloqueada!")

💡 Problema: Senha Fixa

Escreva um programa que repita a leitura de uma senha até que ela seja válida. Para cada leitura de senha incorreta informada, escrever a mensagem "Senha Invalida! Tente novamente:". Quando a senha for informada corretamente deve ser impressa a mensagem "Acesso Permitido" e o algoritmo encerrado. Considere que a senha correta é o valor 2002.
