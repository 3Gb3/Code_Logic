Sistema de Salas:

# Salas disponíveis
salas = ["Entrada", "Corredor", "Sala do Tesouro", "Armadilha", "Saída"]
sala_atual = 0  # Começa na entrada
pontos = 0
vidas = 3

# Itens coletados
inventario = []

# Explorar
while sala_atual < len(salas) and vidas > 0:
    print(f"\n📍 Você está em: {salas[sala_atual]}")
    print(f"💖 Vidas: {vidas} | ⭐ Pontos: {pontos}")
    
    # Ações na sala atual
    if sala_atual == 0:  # Entrada
        print("Você vê duas portas...")
    elif sala_atual == 2:  # Tesouro
        print("💎 Você encontrou um tesouro!")
        inventario.append("Diamante")
        pontos += 100
    
    # Próxima sala
    sala_atual += 1

🎲 Sistema de Desafios

Exemplo de Desafio com Tentativas:

print("\n🔒 PORTA TRANCADA!")
print("Adivinhe o número de 1 a 5 para abrir")

numero_secreto = 3
tentativas = 0
max_tentativas = 3

while tentativas < max_tentativas:
    palpite = int(input(f"Tentativa {tentativas + 1}/{max_tentativas}: "))
    tentativas += 1
    
    if palpite == numero_secreto:
        print("✅ Porta aberta!")
        pontos += 50
        break
    elif palpite < numero_secreto:
        print("📈 Maior!")
    else:
        print("📉 Menor!")
    
    if tentativas == max_tentativas:
        print("❌ Porta permanece trancada!")
        vidas -= 1

📦 Sistema de Inventário

# Mostrar inventário
print("\n🎒 SEU INVENTÁRIO:")
if len(inventario) == 0:
    print("  Vazio")
else:
    for i in range(len(inventario)):
        print(f"  {i+1}. {inventario[i]}")

# Usar item
print("\nItens disponíveis para usar:")
for i in range(len(inventario)):
    print(f"{i+1}. {inventario[i]}")

escolha = int(input("Usar item (0 para cancelar): "))

if escolha > 0 and escolha <= len(inventario):
    item_usado = inventario[escolha - 1]
    print(f"Você usou: {item_usado}")
    inventario.remove(item_usado)

💰 Sistema de Coleta

# Procurar tesouros
tesouros_encontrados = 0
total_salas = 5

print("🔍 PROCURANDO TESOUROS...")

for sala in range(1, total_salas + 1):
    print(f"\nSala {sala}:")
    
    # Simular busca (usuário digita 's' para procurar)
    acao = input("Procurar nesta sala? (s/n): ")
    
    if acao.lower() == 's':
        # Chance de encontrar
        encontrou = sala % 2 == 0  # Salas pares têm tesouro
        
        if encontrou:
            print("💎 Tesouro encontrado!")
            tesouros_encontrados += 1
            pontos += 50
        else:
            print("😔 Nada aqui...")
    else:
        print("⏭️ Pulou a sala")
        continue

print(f"\n✨ Total de tesouros: {tesouros_encontrados}")

🏆 Sistema de Pontuação Final

# Calcular pontuação final
print("\n" + "="*40)
print("🏆 RESULTADO FINAL")
print("="*40)

# Bônus por sobrevivência
bonus_vidas = vidas * 100

# Bônus por itens
bonus_itens = len(inventario) * 50

# Total
pontuacao_final = pontos + bonus_vidas + bonus_itens

print(f"Pontos base: {pontos}")
print(f"Bônus vidas: {bonus_vidas} ({vidas} x 100)")
print(f"Bônus itens: {bonus_itens} ({len(inventario)} x 50)")
print(f"\n⭐ PONTUAÇÃO FINAL: {pontuacao_final}")

# Classificação
if pontuacao_final >= 500:
    print("🥇 LENDÁRIO!")
elif pontuacao_final >= 300:
    print("🥈 HERÓI!")
elif pontuacao_final >= 100:
    print("🥉 AVENTUREIRO!")
else:
    print("📝 INICIANTE")

print("="*40)

💡 Conceitos Utilizados

  • While loops: Menu principal e desafios
  • For loops: Percorrer salas e inventário
  • Break: Sair de desafios quando acertar
  • Continue: Pular salas
  • Listas: Inventário e salas
  • Contadores: Pontos, vidas, tentativas
