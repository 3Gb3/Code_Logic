🎯 Objetivo

  • Gerar sequências de números ímpares
  • Usar range com passo 2
  • Processar múltiplas sequências

📚 Conceitos: Gerando Ímpares

Assim como pares, podemos gerar ímpares de forma eficiente:

Método 1: Range com ímpares

# Gerar ímpares de 1 a 19
for impar in range(1, 20, 2):
    print(impar)  # 1, 3, 5, 7, 9, ..., 19

Método 2: Cálculo direto

# Primeiro ímpar >= X
x = 10
if x % 2 == 0:
    primeiro_impar = x + 1
else:
    primeiro_impar = x

print(primeiro_impar)  # 11

Exemplo - 5 ímpares a partir de X:

x = 12

# Ajustar para próximo ímpar
if x % 2 == 0:
    x = x + 1

# Gerar 5 ímpares
for i in range(5):
    impar = x + (i * 2)
    print(impar)
# Saída: 13, 15, 17, 19, 21

🎯 Padrão: Múltiplas Sequências

Para processar N sequências diferentes:

Estrutura:

n = int(input("Quantos testes? "))

for teste in range(n):
    # Ler parâmetro da sequência
    x = int(input("Valor de X: "))
    
    # Processar esta sequência
    # ...
    
    print()  # Linha em branco entre sequências

Aplicação no problema:

n = int(input("Digite o valor de N: "))

for i in range(n):
    x = int(input("Digite o valor de X: "))
    
    # Ajustar x para o primeiro ímpar >= x
    if x % 2 == 0:
        x = x + 1
    
    # Imprimir x ímpares consecutivos
    for j in range(x):
        impar = x + (j * 2)
        print(impar)
