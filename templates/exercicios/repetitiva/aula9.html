🔍 Busca Linear

Percorre a lista elemento por elemento até encontrar o valor procurado.

Buscar um Valor:

numeros = [64, 34, 25, 12, 22, 11, 90]
valor_procurado = 22
posicao = -1  # -1 significa não encontrado

# Buscar
for i in range(len(numeros)):
    if numeros[i] == valor_procurado:
        posicao = i
        break  # Parar quando encontrar

# Resultado
if posicao != -1:
    print(f"✅ Valor {valor_procurado} encontrado na posição {posicao}")
else:
    print(f"❌ Valor {valor_procurado} não encontrado")

Buscar Todas as Ocorrências:

numeros = [1, 3, 5, 3, 7, 3, 9]
valor = 3
posicoes = []

# Buscar todas
for i in range(len(numeros)):
    if numeros[i] == valor:
        posicoes.append(i)

print(f"Valor {valor} encontrado nas posições: {posicoes}")
# Saída: [1, 3, 5]

📊 Encontrar Maior e Menor

Encontrar o Maior Valor:

numeros = [34, 87, 12, 56, 90, 23]
maior = numeros[0]  # Começa com o primeiro

for num in numeros:
    if num > maior:
        maior = num

print(f"Maior valor: {maior}")

Encontrar Menor e sua Posição:

numeros = [34, 87, 12, 56, 90, 23]
menor = numeros[0]
posicao_menor = 0

for i in range(len(numeros)):
    if numeros[i] < menor:
        menor = numeros[i]
        posicao_menor = i

print(f"Menor valor: {menor} na posição {posicao_menor}")

🔄 Inverter Lista

Criar Lista Invertida:

original = [1, 2, 3, 4, 5]
invertida = []

# Percorrer de trás para frente
for i in range(len(original) - 1, -1, -1):
    invertida.append(original[i])

print(f"Original: {original}")
print(f"Invertida: {invertida}")

Inverter no Lugar (Troca):

lista = [1, 2, 3, 4, 5]
print(f"Antes: {lista}")

# Trocar elementos das pontas
n = len(lista)
for i in range(n // 2):
    # Trocar posição i com posição n-1-i
    temp = lista[i]
    lista[i] = lista[n - 1 - i]
    lista[n - 1 - i] = temp

print(f"Depois: {lista}")

🎯 Remover Duplicatas

numeros = [1, 2, 2, 3, 4, 4, 4, 5, 1]
sem_duplicatas = []

# Adicionar apenas se não existir
for num in numeros:
    if num not in sem_duplicatas:
        sem_duplicatas.append(num)

print(f"Original: {numeros}")
print(f"Sem duplicatas: {sem_duplicatas}")

📈 Contar Frequências

votos = ["A", "B", "A", "C", "B", "A", "B", "B"]

# Contar cada candidato
count_a = 0
count_b = 0
count_c = 0

for voto in votos:
    if voto == "A":
        count_a += 1
    elif voto == "B":
        count_b += 1
    elif voto == "C":
        count_c += 1

print(f"Candidato A: {count_a} votos")
print(f"Candidato B: {count_b} votos")
print(f"Candidato C: {count_c} votos")

# Determinar vencedor
if count_a > count_b and count_a > count_c:
    print("Vencedor: A")
elif count_b > count_a and count_b > count_c:
    print("Vencedor: B")
else:
    print("Vencedor: C")

🔢 Verificar se é Palíndromo

texto = input("Digite uma palavra: ").lower()
eh_palindromo = True

# Comparar caractere por caractere
n = len(texto)
for i in range(n // 2):
    if texto[i] != texto[n - 1 - i]:
        eh_palindromo = False
        break

if eh_palindromo:
    print(f"✅ '{texto}' é um palíndromo!")
else:
    print(f"❌ '{texto}' não é um palíndromo")

💡 Dicas de Algoritmos

  • 🎯 Use break para parar quando encontrar o que procura
  • 📝 Use variáveis auxiliares para guardar resultados temporários
  • 🔄 range() reverso: range(n-1, -1, -1)
  • 📊 Para trocar valores: use variável temporária
  • 🔍 Para verificar se existe: use "in lista"
