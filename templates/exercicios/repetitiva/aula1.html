🔄 O que são Loops?

Loops (ou estruturas de repetição) permitem executar um bloco de código várias vezes, evitando repetição desnecessária no código.

💡 Analogia do Dia a Dia

Imagine que você precisa escovar os dentes:

ENQUANTO não estiver limpo, REPITA:
1. Aplique pasta de dente
2. Escove por 30 segundos
3. Enxágue

🔧 Tipos de Loops em Python

🔢 Loop for

Usado quando sabemos quantas vezes queremos repetir

for i in range(5):
    print(f"Repetição {i}")

⏰ Loop while

Usado quando repetimos enquanto uma condição for verdadeira

contador = 0
while contador < 5:
    print(f"Contador: {contador}")
    contador += 1

🎮 Exemplos Práticos

Exemplo 1: Contagem com for

print("Contagem de 1 a 5:")
for numero in range(1, 6):
    print(numero)

Resultado:

Contagem de 1 a 5:
1
2
3
4
5

Exemplo 2: Tabuada com for

numero = 3
print(f"Tabuada do {numero}:")
for i in range(1, 11):
    resultado = numero * i
    print(f"{numero} x {i} = {resultado}")

Resultado: Tabuada completa do 3

🧮 A função range()

A função range() é muito usada com loops for:

  • range(5) → 0, 1, 2, 3, 4
  • range(1, 6) → 1, 2, 3, 4, 5
  • range(0, 10, 2) → 0, 2, 4, 6, 8

⚠️ Pontos Importantes

  • Indentação: O código dentro do loop deve estar indentado
  • Condição de parada: Todo loop precisa ter um fim
  • Loop infinito: Cuidado para não criar loops que nunca param
  • Performance: Loops podem ser lentos com muitas repetições
