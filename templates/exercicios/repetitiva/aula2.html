🎯 Objetivo

  • Repetir leitura de múltiplos conjuntos de dados
  • Usar condição de parada
  • Processar dados até flag especial

📚 Conceitos: Loop com Condição de Parada

Muitas vezes não sabemos quantas vezes o loop vai repetir. Usamos um valor sentinela para parar:

Padrão com valor sentinela:

# Lê primeiro valor
valor = int(input("Digite um valor (0 para sair): "))

# Processa enquanto não for o valor de parada
while valor != 0:
    print(f"Processando: {valor}")
    valor = int(input("Digite um valor (0 para sair): "))

Exemplo - Soma até digitar 0:

soma = 0
numero = int(input("Digite um numero (0 para terminar): "))

while numero != 0:
    soma += numero
    numero = int(input("Digite um numero (0 para terminar): "))

print(f"Soma total: {soma}")

🎯 Estratégia: Processar Pares de Valores

Quando precisamos processar pares de valores até uma condição:

Padrão:

# Lê primeiro par
x = int(input("X: "))
y = int(input("Y: "))

# Processa enquanto condição não for satisfeita
while condicao_de_parada_nao_satisfeita:
    # Processa o par (x, y)
    print(f"Processando: ({x}, {y})")
    
    # Lê próximo par
    x = int(input("X: "))
    y = int(input("Y: "))

print("Fim!")

Aplicação no problema:

Vamos ler coordenadas (X, Y) e determinar o quadrante até que ambas sejam zero.

�� Problema: Quadrante

Escreva um programa para ler as coordenadas (X,Y) de uma quantidade indeterminada de pontos no sistema cartesiano. Para cada ponto escrever o quadrante a que ele pertence (Q1, Q2, Q3 ou Q4). O algoritmo será encerrado quando pelo menos uma de duas coordenadas for NULA (nesta situação sem escrever mensagem alguma).
