📐 Fórmulas do Retângulo

Para um retângulo de base b e altura h:

1. Área

area = base × altura

2. Perímetro

perimetro = 2 × (base + altura)

O perímetro é a soma de todos os lados.

3. Diagonal

diagonal = √(base² + altura²)

Usamos o teorema de Pitágoras!

🔢 Biblioteca math

Python possui uma biblioteca para operações matemáticas avançadas:

Importando:

import math

Funções úteis:

  • math.sqrt(x) - raiz quadrada de x
  • math.pow(x, y) - x elevado a y
  • math.pi - valor de π (pi)

Exemplo:

import math
diagonal = math.sqrt(base**2 + altura**2)

📊 Potenciação em Python

Para elevar ao quadrado, use o operador **:

base = 4.0
quadrado = base ** 2  # 16.0
