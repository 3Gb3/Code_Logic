🎯 Objetivo

  • Trabalhar com múltiplas entradas de dados
  • Combinar strings e números na saída
  • Calcular média aritmética
  • Formatar saída com 1 casa decimal

📝 Lendo Dados de Múltiplas Pessoas

Quando precisamos ler dados de várias pessoas, organizamos em blocos:

print("Dados da primeira pessoa:")
nome1 = input("Nome: ")
idade1 = int(input("Idade: "))

print("Dados da segunda pessoa:")
nome2 = input("Nome: ")
idade2 = int(input("Idade: "))

Usamos nomes de variáveis diferentes (nome1, nome2, idade1, idade2) para cada pessoa.

📊 Média Aritmética

A média aritmética é a soma dos valores dividida pela quantidade:

media = (idade1 + idade2) / 2

Para 3 valores seria: (a + b + c) / 3

💡 Combinando Strings e Números

Use f-strings para combinar texto e variáveis:

print(f"A idade média de {nome1} e {nome2} é de {media:.1f} anos")

O :.1f formata o número com 1 casa decimal.

🎮 Problema "idades"

Fazer um programa para ler o nome e idade de duas pessoas. Ao final mostrar uma mensagem com os nomes e a idade média entre essas pessoas, com uma casa decimal.

📝 Exemplo:

Dados da primeira pessoa:
Nome: Maria Silva
Idade: 19
Dados da segunda pessoa:
Nome: Joao Melo
Idade: 20
A idade média de Maria Silva e Joao Melo é de 19.5 anos
