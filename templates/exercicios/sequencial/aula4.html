🧮 Operadores Aritméticos

Operadores básicos:

  • + - Adição
  • - - Subtração
  • * - Multiplicação
  • / - Divisão (resultado decimal)
  • // - Divisão inteira (sem decimais)
  • % - Módulo (resto da divisão)
  • ** - Potenciação

Exemplos:

print(10 + 3)   # 13
print(10 - 3)   # 7
print(10 * 3)   # 30
print(10 / 3)   # 3.333...
print(10 // 3)  # 3
print(10 % 3)   # 1
print(10 ** 3)  # 1000

📊 Precedência de Operadores

Python segue a ordem matemática tradicional:

  1. Parênteses: ()
  2. Potenciação: **
  3. Multiplicação, Divisão, Módulo: *, /, //, %
  4. Adição e Subtração: +, -

Exemplos:

print(2 + 3 * 4)        # 14 (não 20)
print((2 + 3) * 4)      # 20
print(2 ** 3 * 4)       # 32
print(2 ** (3 * 4))     # 4096

🔧 Operadores de Atribuição

Shortcuts para operações comuns:

x = 10
x += 5    # x = x + 5  → x = 15
x -= 3    # x = x - 3  → x = 12
x *= 2    # x = x * 2  → x = 24
x /= 4    # x = x / 4  → x = 6.0
x //= 2   # x = x // 2 → x = 3.0
x %= 2    # x = x % 2  → x = 1.0
x **= 3   # x = x ** 3 → x = 1.0

💡 Aplicações Práticas

Calculando área de um círculo:

raio = float(input("Digite o raio: "))
pi = 3.14159
area = pi * raio ** 2
print(f"A área é: {area:.2f}")

Convertendo temperatura:

celsius = float(input("Temperatura em °C: "))
fahrenheit = (celsius * 9/5) + 32
print(f"{celsius}°C = {fahrenheit}°F")
