🎲 Biblioteca Random

A biblioteca random permite gerar números e escolhas aleatórias:

import random

Principais funções:

random() - número decimal entre 0 e 1:

print(random.random())  # Ex: 0.7234567890123456

randint() - número inteiro entre dois valores:

dado = random.randint(1, 6)  # Simula dado de 6 faces
print(f"Dado: {dado}")

uniform() - número decimal entre dois valores:

altura = random.uniform(1.5, 2.0)  # Altura entre 1.5 e 2.0m
print(f"Altura: {altura:.2f}m")

choice() - escolhe um elemento de uma lista:

cores = ["vermelho", "azul", "verde", "amarelo"]
cor_escolhida = random.choice(cores)
print(f"Cor sorteada: {cor_escolhida}")

shuffle() - embaralha uma lista:

cartas = ["A", "K", "Q", "J"]
random.shuffle(cartas)  # Modifica a lista original
print(f"Cartas embaralhadas: {cartas}")

🎮 Exemplos Práticos

Jogo de Adivinhação:

import random

numero_secreto = random.randint(1, 100)
tentativa = int(input("Adivinhe o número (1-100): "))

if tentativa == numero_secreto:
    print("Parabéns! Você acertou!")
elif tentativa < numero_secreto:
    print("O número é maior!")
else:
    print("O número é menor!")
    
print(f"O número era: {numero_secreto}")

Gerador de Senhas:

import random

letras = "abcdefghijklmnopqrstuvwxyz"
numeros = "0123456789"
simbolos = "!@#$%&*"

todos_chars = letras + letras.upper() + numeros + simbolos
senha = ""

for i in range(8):  # Senha de 8 caracteres
    senha += random.choice(todos_chars)
    
print(f"Senha gerada: {senha}")

Simulador de Cara ou Coroa:

import random

resultado = random.choice(["Cara", "Coroa"])
print(f"Resultado: {resultado}")

# Ou usando randint:
moeda = random.randint(0, 1)
resultado = "Cara" if moeda == 0 else "Coroa"
print(f"Resultado: {resultado}")

📊 Estatísticas e Simulações

Simulação de dados:

import random

# Simular 1000 jogadas de dado
resultados = []
for i in range(1000):
    dado = random.randint(1, 6)
    resultados.append(dado)

# Contar frequência de cada face
for face in range(1, 7):
    frequencia = resultados.count(face)
    percentual = (frequencia / 1000) * 100
    print(f"Face {face}: {frequencia} vezes ({percentual:.1f}%)")

Sorteio de nomes:

import random

participantes = ["Ana", "Bruno", "Carlos", "Diana", "Eduardo"]
vencedor = random.choice(participantes)
print(f"Vencedor do sorteio: {vencedor}")

# Para sortear múltiplos vencedores sem repetição:
vencedores = random.sample(participantes, 3)  # 3 vencedores
print(f"Vencedores: {vencedores}")

🔧 Semente (Seed)

Para resultados reproduzíveis, use seed:

import random

random.seed(42)  # Sempre gerará a mesma sequência
print(random.randint(1, 10))  # Sempre será o mesmo número
print(random.randint(1, 10))  # Sempre será o mesmo número

# Sem seed, números são diferentes a cada execução
