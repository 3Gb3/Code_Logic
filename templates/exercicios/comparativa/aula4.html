📚 Conceitos: Comparação de Strings

Podemos comparar strings (textos) usando os mesmos operadores de números:

Exemplo:

opcao = input("Escolha (S/N): ")

if opcao == "S":
    print("Você escolheu SIM")
elif opcao == "N":
    print("Você escolheu NÃO")
else:
    print("Opção inválida")

⚠️ IMPORTANTE:

  • Python diferencia maiúsculas de minúsculas: "S" ≠ "s"
  • Use .upper() ou .lower() para padronizar

Exemplo com conversão:

opcao = input("Escolha (S/N): ").upper()  # Converte para maiúscula
if opcao == "S":
    print("SIM")

🌡️ Fórmulas de Conversão

Celsius para Fahrenheit:

F = (9 × C / 5) + 32

Fahrenheit para Celsius:

C = (F - 32) × 5 / 9

Em Python:

# Celsius → Fahrenheit
celsius = 25.0
fahrenheit = (9 * celsius / 5) + 32
print(f"{celsius}°C = {fahrenheit}°F")

# Fahrenheit → Celsius
fahrenheit = 77.0
celsius = (fahrenheit - 32) * 5 / 9
print(f"{fahrenheit}°F = {celsius}°C")

📊 Referências:

  • 0°C = 32°F (ponto de congelamento da água)
  • 100°C = 212°F (ponto de ebulição da água)
  • 37°C = 98.6°F (temperatura corporal)

💡 Problema: Conversão de Temperatura

Deseja-se converter uma medida de temperatura da escala Celsius para Fahrenheit ou vice-versa. Para isso, você deve construir um programa que leia a letra "C" ou "F" indicando em qual escala vai ser informada uma temperatura. Em seguida o programa deve mostrar a temperatura na outra escala com duas casas decimais.
