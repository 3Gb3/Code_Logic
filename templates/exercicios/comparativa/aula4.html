💡 Operadores Lógicos em Python

Os operadores lógicos permitem combinar ou modificar condições:

🔗 Operador and

Retorna True apenas se AMBAS as condições forem verdadeiras.

idade >= 18 and tem_carteira == True

🔀 Operador or

Retorna True se PELO MENOS UMA das condições for verdadeira.

dia == "sábado" or dia == "domingo"

🚫 Operador not

Inverte o valor lógico (True vira False e vice-versa).

not chovendo

Exemplo 1: Sistema de Login

senha = input("Senha: ")

if usuario == "admin" and senha == "123456":
    print("Login realizado com sucesso!")
else:
    print("Usuário ou senha incorretos.")

Exemplo 2: Desconto em Loja

idade = int(input("Sua idade: "))
eh_estudante = input("É estudante? (s/n): ") == "s"

if idade >= 60 or eh_estudante:
    print("Você tem direito a 15% de desconto!")

    print("Sem desconto disponível.")
