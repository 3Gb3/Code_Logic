📚 Conceitos: Estruturas Condicionais

As estruturas condicionais permitem que o programa tome decisões:

Sintaxe básica:

if condicao:
    # código executado se condição for True
else:
    # código executado se condição for False

Operadores de comparação:

  • == igual a
  • != diferente de
  • > maior que
  • < menor que
  • >= maior ou igual
  • <= menor ou igual

Exemplo:

idade = int(input("Digite sua idade: "))
if idade >= 18:
    print("Você é maior de idade")
else:
    print("Você é menor de idade")

📐 Fórmula de Baskara

A fórmula de Baskara resolve equações do segundo grau: ax² + bx + c = 0

Passo 1: Calcular o delta (Δ)

Δ = b² - 4ac

Passo 2: Verificar se há raízes reais

  • Se Δ < 0: não há raízes reais
  • Se Δ ≥ 0: há raízes reais

Passo 3: Calcular as raízes (se Δ ≥ 0)

x1 = (-b + √Δ) / (2a)
x2 = (-b - √Δ) / (2a)

Em Python:

import math

delta = b**2 - 4*a*c
if delta < 0:
    print("Esta equacao nao possui raizes reais")
else:
    x1 = (-b + math.sqrt(delta)) / (2 * a)
    x2 = (-b - math.sqrt(delta)) / (2 * a)

💡 Problema: Fórmula de Baskara

Fazer um programa para ler os três coeficientes de uma equação do segundo grau. Usando a fórmula de Baskara, calcular e mostrar o valor das raízes x1 e x2 da equação com cinco casas decimais. Se a equação não possuir raízes reais, mostrar uma mensagem.

Casos especiais:

  • Se a = 0: "Esta equacao nao e do segundo grau"
  • Se Δ < 0: "Esta equacao nao possui raizes reais"
