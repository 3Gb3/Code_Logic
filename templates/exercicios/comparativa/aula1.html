💡 O que são Estruturas Condicionais?

Estruturas condicionais permitem que nossos programas tomem decisões baseadas em condições específicas.

É como se o programa pudesse "pensar": "Se isso for verdade, faça aquilo; caso contrário, faça outra coisa"

Exemplo do mundo real: "Se estiver chovendo, leve um guarda-chuva; caso contrário, não precisa levar."

🔧 A Estrutura if

A palavra-chave if é usada para executar código apenas quando uma condição é verdadeira.

Sintaxe Básica:

if condição:
    # código que executa se a condição for verdadeira
    print("A condição é verdadeira!")

Exemplo Prático:

idade = 18
if idade >= 18:
    print("Você é maior de idade!")

Pontos importantes:

  • Use dois pontos : após a condição
  • O código dentro do if deve estar indentado (4 espaços)
  • A condição deve resultar em True ou False

⚖️ A Estrutura if/else

O else permite executar código quando a condição do if é falsa.

Sintaxe:

if condição:
    # executa se condição for verdadeira
    print("Condição verdadeira")
else:
    # executa se condição for falsa
    print("Condição falsa")

Exemplo Completo:

idade = int(input("Digite sua idade: "))

if idade >= 18:
    print("Você pode dirigir!")
else:
    print("Você ainda não pode dirigir.")

🌡️ Exemplos Práticos

Exemplo 1: Verificador de Temperatura

temperatura = 25

if temperatura > 30:
    print("Está muito quente!")
else:
    print("Temperatura agradável.")

Exemplo 2: Verificador de Nota

nota = float(input("Digite sua nota: "))

if nota >= 7.0:
    print("Parabéns! Você foi aprovado!")
else:
    print("Estude mais e tente novamente.")

Exemplo 3: Par ou Ímpar

numero = int(input("Digite um número: "))

if numero % 2 == 0:
    print("O número é par")
else:
    print("O número é ímpar")

🎮 Vamos praticar!

Desafio: Crie um programa que funcione como um "detector de idade para filmes".

Requisitos:

  1. Solicite a idade do usuário
  2. Se a idade for maior ou igual a 18, exiba: "Você pode assistir filmes para maiores!"
  3. Caso contrário, exiba: "Você só pode assistir filmes livres!"
  4. Teste com diferentes idades

Dicas:

  • Use int(input("...")) para ler um número inteiro
  • Lembre-se da indentação (4 espaços)
  • Use dois pontos : após if e else
