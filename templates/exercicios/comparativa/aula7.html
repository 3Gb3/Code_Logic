⚡ O que são Expressões Condicionais?

Também conhecidas como "operador ternário", permitem escrever condições simples em uma única linha.

Sintaxe:

variavel = valor_se_true if condicao else valor_se_false

Equivale a:

if condicao:
    variavel = valor_se_true
else:
    variavel = valor_se_false

📝 Exemplos Básicos

Exemplo 1: Determinando Status

# Forma tradicional
idade = 20
if idade >= 18:
    status = "Maior de idade"
else:
    status = "Menor de idade"

# Com expressão condicional
status = "Maior de idade" if idade >= 18 else "Menor de idade"
print(status)  # Maior de idade

Exemplo 2: Calculando Desconto

valor = 100
eh_vip = True

# Aplicar desconto de 20% se for VIP, senão 5%
desconto = 0.20 if eh_vip else 0.05
valor_final = valor * (1 - desconto)

print(f"Valor final: R$ {valor_final}")  # R$ 80.0

🔢 Expressões Condicionais com Números

# Encontrar o maior número
a = 10
b = 15
maior = a if a > b else b
print(f"Maior número: {maior}")

# Determinar paridade
numero = 7
tipo = "par" if numero % 2 == 0 else "ímpar"
print(f"{numero} é {tipo}")

# Nota mínima para aprovação
nota = 8.5
resultado = "Aprovado" if nota >= 7 else "Reprovado"
print(resultado)

📚 Expressões Condicionais Aninhadas

⚠️ Use com moderação para manter legibilidade!

# Classificação de notas
nota = 8.5
conceito = ("A" if nota >= 9 else 
           "B" if nota >= 7 else 
           "C" if nota >= 5 else "D")
print(f"Conceito: {conceito}")

# Melhor usar if-elif para casos complexos
if nota >= 9:
    conceito = "A"
elif nota >= 7:
    conceito = "B"
elif nota >= 5:
    conceito = "C"
else:
    conceito = "D"

🎨 Casos de Uso Práticos

1. Formatação de Texto

produtos = 3
mensagem = f"Você tem {produtos} produto{'s' if produtos != 1 else ''} no carrinho"
print(mensagem)

2. Valores Padrão

nome = input("Seu nome: ")
nome_final = nome if nome.strip() else "Usuário Anônimo"
print(f"Olá, {nome_final}!")

3. Cálculos Condicionais

salario = 3000
eh_funcionario = True
bonus = salario * 0.1 if eh_funcionario else 0
print(f"Bônus: R$ {bonus}")

⚖️ Quando Usar Expressões Condicionais

✅ Use quando:

  • A condição é simples e clara
  • Há apenas dois resultados possíveis
  • A linha não fica muito longa
  • Melhora a legibilidade do código

❌ Evite quando:

  • A condição é complexa
  • Há múltiplas condições aninhadas
  • A linha fica muito longa
  • Prejudica a legibilidade

💡 Dicas de Boas Práticas

  • 📏 Mantenha expressões curtas e simples
  • 🔤 Use parênteses para agrupar condições complexas
  • 📖 Priorize sempre a legibilidade
  • 🧪 Teste com diferentes valores
  • 💭 Considere usar if tradicional para lógica complexa
