🎯 Objetivo

  • Entender o conceito de if aninhados
  • Organizar condições complexas
  • Evitar código confuso e melhorar legibilidade

🏗️ O que são Condições Aninhadas?

São estruturas if-else colocadas dentro de outras estruturas if-else, criando múltiplos níveis de decisão.

Estrutura Básica:

if condicao1:
    if condicao2:
        # Executado se ambas forem True
        print("Condições 1 e 2 são verdadeiras")
    else:
        # Executado se apenas condicao1 for True
        print("Apenas condição 1 é verdadeira")
else:
    print("Condição 1 é falsa")

📝 Exemplo Prático: Sistema de Notas

nota = float(input("Digite sua nota: "))

if nota >= 7:
    print("Aprovado!")
    if nota >= 9:
        print("Parabéns! Excelente desempenho!")
    elif nota >= 8:
        print("Muito bom!")
    else:
        print("Bom trabalho!")
else:
    print("Reprovado.")
    if nota >= 5:
        print("Você pode fazer recuperação.")
    else:
        print("Precisa refazer a matéria.")

⚖️ Quando Usar vs Operadores Lógicos

✅ Use Condições Aninhadas quando:

  • As condições internas dependem da externa
  • Cada nível precisa de ações diferentes
  • A lógica fica mais clara dividida em etapas

✅ Use Operadores Lógicos quando:

  • Todas as condições são independentes
  • A ação é a mesma para diferentes combinações
  • A condição é simples

🔄 Comparação: Aninhado vs Lógico

Versão com Operadores Lógicos:

if idade >= 18 and tem_carteira and tem_carro:
    print("Pode dirigir sozinho")

Versão Aninhada (mais detalhada):

if idade >= 18:
    print("Maior de idade ✓")
    if tem_carteira:
        print("Tem carteira ✓")
        if tem_carro:
            print("Pode dirigir sozinho!")
        else:
            print("Precisa de um carro")
    else:
        print("Precisa tirar carteira")
else:
    print("Menor de idade")

🎮 Exercício Prático

Sistema de Cadastro de Usuário

Crie um sistema que valide o cadastro com condições aninhadas:

  • Se idade ≥ 18: pode se cadastrar
  • Se tem email: verificar se tem @
  • Se senha tem ≥ 8 caracteres: cadastro completo
  • Senão: pedir senha mais forte

💡 Dicas Importantes

  • 🎯 Mantenha no máximo 3 níveis de aninhamento
  • 📏 Use indentação correta (4 espaços por nível)
  • 💭 Comente código complexo
  • 🔄 Considere usar elif quando possível
