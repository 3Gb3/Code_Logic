📚 Conceitos: Faixas de Valores

Quando temos regras diferentes para faixas de valores, usamos comparações encadeadas:

Padrão de faixas:

if valor <= limite1:
    # Faixa 1
elif valor <= limite2:
    # Faixa 2
elif valor <= limite3:
    # Faixa 3
else:
    # Faixa 4

Exemplo - Desconto progressivo:

total = 1500

if total <= 500:
    desconto = 0.05  # 5%
elif total <= 1000:
    desconto = 0.10  # 10%
elif total <= 2000:
    desconto = 0.15  # 15%
else:
    desconto = 0.20  # 20%

valor_desconto = total * desconto
valor_final = total - valor_desconto

De R$ 3000.01 a R$ 8000.0010%
Acima de R$ 8000.005%

Cálculos:

  • Aumento: salario × (percentual / 100)
  • Novo salário: salario + aumento

Exemplo:

salario = 2500.00
percentual = 15  # 15% para salários entre 1000 e 3000

aumento = salario * (percentual / 100)  # 375.00
novo_salario = salario + aumento  # 2875.00

💡 Problema: Aumento Salarial

Uma empresa vai conceder um aumento percentual de salário aos seus funcionários dependendo de quanto cada pessoa ganha. Fazer um programa para ler o salário de uma pessoa, daí mostrar qual o novo salário desta pessoa depois do aumento, quanto foi o aumento e qual foi a porcentagem de aumento.
