🎯 Objetivo

  • Escrever código condicional limpo e legível
  • Usar nomes de variáveis descritivos
  • Organizar condições de forma clara

✨ Nomes Descritivos

❌ Código Ruim:

x = int(input("Digite: "))
if x > 18:
    y = True
    print(y)

✅ Código Bom:

idade = int(input("Digite sua idade: "))
if idade > 18:
    pode_votar = True
    print("Você pode votar!")

Por quê? Nomes claros facilitam a leitura!

🎯 Condições Diretas

❌ Redundante:

idade = int(input("Idade: "))
if idade >= 18:
    pode_votar = True
else:
    pode_votar = False
print(pode_votar)

✅ Direto:

idade = int(input("Idade: "))
pode_votar = idade >= 18
print(f"Pode votar: {pode_votar}")

Dica: Quando uma condição retorna True/False, atribua diretamente!

🔍 Validação em Etapas

❌ Aninhamento Excessivo:

nome = input("Nome: ")
idade = int(input("Idade: "))
email = input("Email: ")

if len(nome) >= 3:
    if idade >= 18:
        if "@" in email:
            print("Cadastro aprovado!")
        else:
            print("Email inválido")
    else:
        print("Idade insuficiente")
else:
    print("Nome muito curto")

✅ Validação Sequencial:

nome = input("Nome: ")
idade = int(input("Idade: "))
email = input("Email: ")

# Validar um de cada vez
if len(nome) < 3:
    print("❌ Nome muito curto")
elif idade < 18:
    print("❌ Idade insuficiente")
elif "@" not in email:
    print("❌ Email inválido")
else:
    print("✅ Cadastro aprovado!")

� Condições Compostas

Múltiplas Verificações:

usuario = input("Usuário: ")
senha = input("Senha: ")

# Verificar tudo de uma vez
usuario_correto = usuario == "admin"
senha_correta = senha == "1234"
tamanho_ok = len(senha) >= 4

if usuario_correto and senha_correta and tamanho_ok:
    print("✅ Login realizado!")
else:
    print("❌ Acesso negado!")
    
    # Mostrar o que está errado
    if not usuario_correto:
        print("- Usuário incorreto")
    if not senha_correta:
        print("- Senha incorreta")
    if not tamanho_ok:
        print("- Senha muito curta")

🎯 Sistema de Notas Limpo

nome = input("Nome do aluno: ")
nota1 = float(input("Nota 1: "))
nota2 = float(input("Nota 2: "))
nota3 = float(input("Nota 3: "))

# Calcular média
media = (nota1 + nota2 + nota3) / 3

# Determinar situação
if media >= 7:
    situacao = "Aprovado"
    emoji = "🎉"
elif media >= 5:
    situacao = "Recuperação"
    emoji = "⚠️"
else:
    situacao = "Reprovado"
    emoji = "❌"

# Exibir resultado organizado
print()
print("="*40)
print(f"BOLETIM DE {nome.upper()}")
print("="*40)
print(f"Nota 1: {nota1:.1f}")
print(f"Nota 2: {nota2:.1f}")
print(f"Nota 3: {nota3:.1f}")
print(f"Média: {media:.2f}")
print(f"Situação: {situacao} {emoji}")
print("="*40)

💡 Dicas de Organização

  • ✅ Use nomes que explicam o que a variável guarda
  • ✅ Evite muitos if dentro de if (máximo 2 níveis)
  • ✅ Agrupe validações relacionadas
  • ✅ Use elif para condições excludentes
  • ✅ Comente apenas o que não é óbvio
  • ✅ Deixe espaço entre seções do código
