📊 Operadores de Comparação

Os operadores de comparação são usados para comparar valores e sempre retornam True ou False.

OperadorSignificadoExemploResultado
==Igual a5 == 5True
!=Diferente de5 != 3True
>Maior que8 > 5True
<Menor que3 < 7True
>=Maior ou igual5 >= 5True
<=Menor ou igual4 <= 6True

🔍 Exemplos Práticos

Comparando Números:

a = 10
b = 15

print(a == b)    # False
print(a != b)    # True
print(a > b)     # False
print(a < b)     # True
print(a >= 10)   # True
print(b <= 20)   # True

Comparando Strings:

nome1 = "Ana"
nome2 = "Bruno"

print(nome1 == nome2)    # False
print(nome1 != nome2)    # True
print(nome1 < nome2)     # True (ordem alfabética)

🎮 Usando em Condições

Exemplo: Sistema de Notas

nota = 8.5

if nota >= 9.0:
    print("Excelente!")
else:
    if nota >= 7.0:
        print("Bom!")
    else:
        print("Precisa melhorar")

Exemplo: Verificação de Senha

senha_correta = "1234"
senha_digitada = "1234"

if senha_digitada == senha_correta:
    print("Acesso liberado!")
else:
    print("Senha incorreta!")

⚠️ Cuidados Importantes

  • == vs =: Use == para comparar, = para atribuir
  • Tipos compatíveis: Compare valores do mesmo tipo quando possível
  • Strings: Comparação é case-sensitive ("A" != "a")
  • Números: Cuidado ao comparar int com float

🏋️‍♂️ Exercício Prático

Crie um programa que compara duas idades e informa qual é maior

📋 Instruções:

  1. Crie duas variáveis: idade1 = 25 e idade2 = 30
  2. Compare as idades usando operadores de comparação
  3. Se idade1 for maior, imprima "A primeira pessoa é mais velha"
  4. Se idade2 for maior, imprima "A segunda pessoa é mais velha"
  5. Se forem iguais, imprima "As duas pessoas têm a mesma idade"

💡 Dicas:

  • Use if, elif e else para cobrir todos os casos
  • Lembre-se dos operadores >, < e ==
  • Teste alterando os valores das idades

📤 Resultado Esperado:

A segunda pessoa é mais velha
