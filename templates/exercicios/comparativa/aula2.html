📚 Conceitos: Condicionais Encadeadas (elif)

Quando precisamos testar múltiplas condições, usamos elif (else if):

Sintaxe:

if condicao1:
    # executa se condicao1 for True
elif condicao2:
    # executa se condicao1 for False e condicao2 for True
else:
    # executa se todas as condições forem False

Operadores Lógicos:

  • and - E lógico (ambas condições devem ser True)
  • or - OU lógico (pelo menos uma condição deve ser True)
  • not - NÃO lógico (inverte a condição)

Exemplo:

nota = float(input("Digite a nota: "))

if nota >= 9.0:
    print("Excelente!")
elif nota >= 7.0:
    print("Bom")
elif nota >= 5.0:
    print("Regular")
else:
    print("Insuficiente")

🎯 Estratégia: Encontrar o Menor

Para encontrar o menor de três valores, comparamos dois a dois:

Método 1: Comparações encadeadas

if valor1 <= valor2 and valor1 <= valor3:
    menor = valor1
elif valor2 <= valor3:
    menor = valor2
else:
    menor = valor3

Método 2: Função min() do Python

menor = min(valor1, valor2, valor3)

💡 Dica: A função min() é mais simples, mas é importante entender a lógica das comparações!
