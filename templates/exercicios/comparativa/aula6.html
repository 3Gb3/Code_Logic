🎯 Objetivo

  • Implementar menu de opções
  • Trabalhar com tabela de preços
  • Calcular total baseado em código e quantidade

📚 Conceitos: Estrutura de Decisão Múltipla

Quando temos múltiplas opções baseadas em um mesmo valor, usamos if/elif/else:

Exemplo - Menu:

codigo = int(input("Código do produto: "))

if codigo == 1:
    preco = 5.00
elif codigo == 2:
    preco = 3.50
elif codigo == 3:
    preco = 4.80
elif codigo == 4:
    preco = 8.90
elif codigo == 5:
    preco = 7.32
else:
    print("Codigo invalido!")
    preco = 0

💡 Alternativa - Dicionário:

precos = {
    1: 5.00,
    2: 3.50,
    3: 4.80,
    4: 8.90,
    5: 7.32
}

codigo = int(input("Código: "))
preco = precos.get(codigo, 0)  # 0 se código inválido
