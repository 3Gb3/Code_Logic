🎯 Objetivo

  • Validar entrada de dados do usuário
  • Tratar erros de digitação e formato
  • Criar sistemas mais robustos e seguros

🛡️ Por que Validar Dados?

A validação de dados é essencial para:

  • 🔒 Garantir segurança do sistema
  • 🎯 Evitar erros e crashes
  • ✅ Manter qualidade dos dados
  • 😊 Melhorar experiência do usuário

📊 Tipos de Validação

1. Validação de Intervalo

idade = int(input("Digite sua idade: "))

if idade >= 0 and idade <= 120:
    print(f"Idade válida: {idade}")
else:
    print("Idade deve estar entre 0 e 120 anos")

2. Validação de Formato

email = input("Digite seu email: ")

# Verificar se tem @ e .
if "@" in email and "." in email:
    # Verificar posições
    pos_arroba = email.find("@")
    pos_ponto = email.find(".", pos_arroba)
    
    if pos_arroba > 0 and pos_ponto > pos_arroba + 1:
        print("Email válido!")
    else:
        print("Formato de email inválido")
else:
    print("Email deve conter @ e .")

3. Validação de Tamanho

nome = input("Digite seu nome: ")
tamanho = len(nome)

if tamanho >= 3 and tamanho <= 50:
    print(f"Nome válido: {nome}")
else:
    print("Nome deve ter entre 3 e 50 caracteres")

🔐 Validação de Senha Simples

senha = input("Digite sua senha: ")
tamanho_senha = len(senha)

# Verificar tamanho mínimo
if tamanho_senha >= 8:
    print("Senha forte! ✅")
    print(f"Sua senha tem {tamanho_senha} caracteres")
else:
    print("Senha fraca! ❌")
    print("Senha deve ter pelo menos 8 caracteres")
    faltam = 8 - tamanho_senha
    print(f"Faltam {faltam} caracteres")

📞 Validação de Telefone

telefone = input("Digite seu telefone (apenas números): ")
tamanho_tel = len(telefone)

# Verificar se só tem números
if telefone.isdigit():
    # Verificar tamanho (10 ou 11 dígitos)

🎮 Validação de Nota

nota = float(input("Digite a nota (0-10): "))

if nota >= 0 and nota <= 10:
    print(f"Nota registrada: {nota}")
    
    # Feedback adicional
    if nota >= 7:
        print("Aprovado! 🎉")
    elif nota >= 5:
        print("Recuperação ⚠️")
    else:
        print("Reprovado ❌")
else:
    print("Nota inválida! Deve estar entre 0 e 10")
