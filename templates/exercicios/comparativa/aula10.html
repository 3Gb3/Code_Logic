🏗️ Visão Geral do Projeto

Vamos criar um Sistema de Boletim Escolar que calcula médias, determina aprovação/reprovação e classifica o desempenho do aluno.

📋 O que o sistema deve fazer:

  • 📊 Coletar nome e 4 notas do aluno
  • 🧮 Calcular média aritmética
  • ✅ Determinar se está aprovado, em recuperação ou reprovado
  • 🏆 Classificar desempenho (Excelente, Bom, Regular, Insuficiente)
  • 📝 Dar feedback personalizado

📏 Critérios de Avaliação

Sistema de Classificação:

  • Média >= 9.0: Excelente 🏆
  • Média >= 7.0: Bom 😊
  • Média >= 6.0: Regular 😐
  • Média < 6.0: Insuficiente 😔

Sistema de Aprovação:

  • Média >= 7.0: APROVADO ✅
  • 5.0 <= Média < 7.0: RECUPERAÇÃO ⚠️
  • Média < 5.0: REPROVADO ❌

🧮 Exemplo de Cálculo

nome = input("Nome do aluno: ")
nota1 = float(input("Nota 1: "))
nota2 = float(input("Nota 2: "))
nota3 = float(input("Nota 3: "))
nota4 = float(input("Nota 4: "))

# Calcular média
media = (nota1 + nota2 + nota3 + nota4) / 4

# Determinar situação
if media >= 7.0:
    situacao = "APROVADO"
    emoji = "✅"
elif media >= 5.0:
    situacao = "RECUPERAÇÃO"
    emoji = "⚠️"
else:
    situacao = "REPROVADO"
    emoji = "❌"

# Classificar desempenho
if media >= 9.0:
    desempenho = "Excelente"
    medalha = "🏆"
elif media >= 7.0:
    desempenho = "Bom"
    medalha = "😊"
elif media >= 6.0:
    desempenho = "Regular"
    medalha = "😐"
else:
    desempenho = "Insuficiente"
    medalha = "😔"

# Exibir boletim
print()
print("="*50)
print(f"BOLETIM ESCOLAR - {nome.upper()}")
print("="*50)
print(f"Nota 1: {nota1:.1f}")
print(f"Nota 2: {nota2:.1f}")
print(f"Nota 3: {nota3:.1f}")
print(f"Nota 4: {nota4:.1f}")
print(f"Média Final: {media:.2f}")
print(f"Desempenho: {desempenho} {medalha}")
print(f"Situação: {situacao} {emoji}")
print("="*50)

🎯 Feedback Personalizado

# Adicionar mensagem personalizada
if media >= 9.0:
    print("🎉 Parabéns! Desempenho excepcional!")
    print("Continue assim!")
elif media >= 7.0:
    print("😊 Bom trabalho!")
    print("Com mais dedicação pode alcançar excelência!")
elif media >= 6.0:
    print("😐 Desempenho regular.")
    print("Precisa estudar mais para melhorar!")
elif media >= 5.0:
    print("⚠️ Atenção! Você está em recuperação.")
    print("Estude mais para a prova de recuperação!")
else:
    print("❌ Infelizmente você foi reprovado.")
    print("Não desanime! Estude mais no próximo ano!")

🔍 Conceitos Aplicados

  • Input: Coleta de dados do usuário

  • Cálculos: Média aritmética
  • If/Elif/Else: Múltiplas classificações
  • Operadores: Comparações e lógica
  • Formatação: f-strings com :.2f
