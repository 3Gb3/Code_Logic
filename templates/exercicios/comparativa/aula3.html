💡 O que é elif?

A palavra-chave elif (abreviação de "else if") permite testar múltiplas condições em sequência.

É muito útil quando você tem várias possibilidades mutuamente exclusivas.

Exemplo do mundo real: "Se a nota for maior que 9, é excelente; senão, se for maior que 7, é bom; senão, se for maior que 5, é regular; senão, é insuficiente."

🔧 Sintaxe do elif

if condição1:
    # código se condição1 for verdadeira
elif condição2:
    # código se condição2 for verdadeira
elif condição3:
    # código se condição3 for verdadeira
else:
    # código se nenhuma condição for verdadeira

📝 Exemplo Prático: Sistema de Notas

nota = float(input("Digite sua nota: "))

if nota >= 9:
    print("Excelente! 🌟")
elif nota >= 7:
    print("Bom trabalho! 👍")
elif nota >= 5:
    print("Regular, pode melhorar 📚")
else:
    print("Precisa estudar mais 💪")

⚠️ Pontos Importantes

  • O Python testa as condições em ordem
  • Quando uma condição é verdadeira, as outras não são testadas
  • Você pode ter quantos elif quiser
  • O else é sempre opcional
