üéØ Padr√£o: Processar Cada Linha

Para processar cada linha e gerar resultado:

Estrutura:

# Matriz MxN (M linhas, N colunas)
m = 3  # linhas
n = 4  # colunas

# Ler matriz
matriz = []
for i in range(m):
    linha = []
    for j in range(n):
        valor = int(input(f"[{i},{j}]: "))
        linha.append(valor)
    matriz.append(linha)

# Processar cada linha
resultados = []
for i in range(m):
    # Processar linha i
    maior = max(matriz[i])
    resultados.append(maior)

# Exibir resultados
print("MAIOR ELEMENTO DE CADA LINHA:")
for resultado in resultados:
    print(resultado)
