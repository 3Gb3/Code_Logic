‚ûï Soma de Matrizes

A soma de matrizes √© feita elemento por elemento. As matrizes devem ter as mesmas dimens√µes.

Exemplo de Soma:

matriz_a = [
    [1, 2, 3],
    [4, 5, 6]
]

matriz_b = [
    [7, 8, 9],
    [10, 11, 12]
]

# Criar matriz resultado
linhas = len(matriz_a)
colunas = len(matriz_a[0])
resultado = []

for i in range(linhas):
    linha_resultado = []
    for j in range(colunas):
        soma = matriz_a[i][j] + matriz_b[i][j]
        linha_resultado.append(soma)
    resultado.append(linha_resultado)

print(resultado)
# [[8, 10, 12], [14, 16, 18]]

üîÑ Transposi√ß√£o de Matriz

A transposi√ß√£o transforma linhas em colunas e vice-versa.

Implementa√ß√£o da Transposi√ß√£o:

    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Criar matriz transposta
linhas = len(matriz)
colunas = len(matriz[0])
transposta = []

for j in range(colunas):
    linha_transposta = []
    for i in range(linhas):
        linha_transposta.append(matriz[i][j])
    transposta.append(linha_transposta)

print("Original:")
for linha in matriz:
    print(linha)

print("\nTransposta:")
for linha in transposta:
    print(linha)

Matriz Identidade:

def criar_identidade(tamanho):
    identidade = []
    for i in range(tamanho):
        linha = []
        for j in range(tamanho):
            if i == j:
                linha.append(1)
            else:
                linha.append(0)
        identidade.append(linha)
    return identidade

matriz_id = criar_identidade(3)
print(matriz_id)
# [[1, 0, 0], [0, 1, 0], [0, 0, 1]]

def criar_zeros(linhas, colunas):

üßÆ Determinante (Matriz 2x2)

C√°lculo do Determinante:

def determinante_2x2(matriz):
    if len(matriz) != 2 or len(matriz[0]) != 2:
        return "Matriz deve ser 2x2"
    
    # det = ad - bc
    a = matriz[0][0]
    b = matriz[0][1]
    c = matriz[1][0]
    d = matriz[1][1]
    
    return a * d - b * c

matriz = [
    [3, 8],
    [4, 6]
]

det = determinante_2x2(matriz)
print(f"Determinante: {det}")  # -14
