üìö Conceitos: Soma de Linhas

Calcular a soma dos elementos de cada linha:

M√©todo 1: Fun√ß√£o sum()

matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Somar linha 0
soma_linha_0 = sum(matriz[0])  # 6

# Somar todas as linhas
for i in range(len(matriz)):
    soma = sum(matriz[i])
    print(f"Soma da linha {i}: {soma}")

M√©todo 2: Loop manual

for i in range(len(matriz)):
    soma = 0
    for j in range(len(matriz[i])):
        soma += matriz[i][j]
    print(f"Linha {i}: {soma}")

üìä Vetor de Resultados

Armazenar o resultado de cada linha em um vetor separado:

Padr√£o:

# Ler matriz
m = int(input("Linhas? "))
n = int(input("Colunas? "))

matriz = []
for i in range(m):
    linha = []
    for j in range(n):
        valor = float(input())
        linha.append(valor)
    matriz.append(linha)

# Criar vetor de somas
vetor_soma = []
for i in range(m):
    soma = sum(matriz[i])
    vetor_soma.append(soma)

# Exibir resultados
print("VETOR GERADO:")
for soma in vetor_soma:
    print(f"{soma:.1f}")
