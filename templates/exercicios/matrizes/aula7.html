A soma de matrizes √© feita elemento por elemento (posi√ß√£o a posi√ß√£o):

Regra da Soma:

C[i][j] = A[i][j] + B[i][j]

Exemplo visual:

Matriz A:      Matriz B:      Matriz C (A+B):
[1  2]    +    [5  6]    =    [6   8]

C[0][0] = A[0][0] + B[0][0] = 1 + 5 = 6
C[0][1] = A[0][1] + B[0][1] = 2 + 6 = 8
C[1][0] = A[1][0] + B[1][0] = 3 + 7 = 10
C[1][1] = A[1][1] + B[1][1] = 4 + 8 = 12

‚ö†Ô∏è Condi√ß√£o:

As duas matrizes DEVEM ter as mesmas dimens√µes (mesmo M e N)

üíª Implementa√ß√£o: Soma de Matrizes

Para somar duas matrizes:

Passo a passo:

# 1. Ler dimens√µes
m = int(input("Linhas? "))
n = int(input("Colunas? "))

# 2. Ler matriz A
print("Digite os valores da matriz A:")
matrizA = []
for i in range(m):
    linha = []
    for j in range(n):
        valor = int(input(f"A[{i},{j}]: "))
        linha.append(valor)
    matrizA.append(linha)

# 3. Ler matriz B
print("Digite os valores da matriz B:")
matrizB = []
for i in range(m):
    linha = []
    for j in range(n):
        valor = int(input(f"B[{i},{j}]: "))
        linha.append(valor)
    matrizB.append(linha)

# 4. Criar matriz C (soma)
matrizC = []
for i in range(m):
    linha = []
    for j in range(n):
        soma = matrizA[i][j] + matrizB[i][j]
        linha.append(soma)
    matrizC.append(linha)

# 5. Exibir matriz C
print("MATRIZ SOMA:")
for i in range(m):
    for j in range(n):
        print(matrizC[i][j], end=" ")
    print()
