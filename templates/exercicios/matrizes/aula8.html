üíª Implementa√ß√£o: Transposta

Para criar a matriz transposta:

n = 2

    [1, 2],
    [3, 4],
    [5, 6]
]

# Transposta: N linhas, M colunas

    linha_nova = []
    for i in range(m):  # Para cada linha original

    transposta.append(linha_nova)

# Resultado: transposta tem 2 linhas e 3 colunas
# [[1, 3, 5], [2, 4, 6]]

Forma alternativa:
