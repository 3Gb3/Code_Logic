üìö Conceitos: Matrizes Retangulares

Matrizes n√£o precisam ser quadradas - podem ter M linhas e N colunas diferentes:

Matriz 2x3 (2 linhas, 3 colunas):

    [1, 2, 3],
    [4, 5, 6]
]

# M=2 linhas, N=3 colunas

Percorrer matriz MxN:

m = 2  # linhas
n = 3  # colunas

# Ler matriz
matriz = []
for i in range(m):
    linha = []
    for j in range(n):
        valor = int(input())
        linha.append(valor)
    matriz.append(linha)

# Percorrer todos os elementos
for i in range(m):
    for j in range(n):
        print(f"[{i}][{j}] = {matriz[i][j]}")

M√©todo 2: List comprehension

    for elem in linha:
        if elem < 0:
            negativos += 1
