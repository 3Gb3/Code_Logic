üìê Somar Regi√£o Espec√≠fica

Para somar apenas elementos acima da diagonal:

Implementa√ß√£o:

matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

soma = 0
for i in range(n):
    for j in range(n):
        if i < j:  # Acima da diagonal
            soma += matriz[i][j]

print(f"Soma acima da diagonal: {soma}")
# Elementos: 2, 3, 6 ‚Üí Soma = 11
