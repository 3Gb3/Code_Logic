üìö Conceitos: Matrizes em Python

Matrizes s√£o listas de listas - estruturas bidimensionais com linhas e colunas:

Criar matriz vazia:

# Matriz 3x3 com zeros
n = 3
matriz = []
for i in range(n):
    linha = [0] * n
    matriz.append(linha)

# Ou de forma mais compacta:
matriz = [[0] * n for i in range(n)]

Acessar elementos:

matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Acessar elemento [linha][coluna]
print(matriz[0][0])  # 1 (primeira linha, primeira coluna)
print(matriz[1][2])  # 6 (segunda linha, terceira coluna)
print(matriz[2][1])  # 8 (terceira linha, segunda coluna)

Percorrer matriz:

# Percorrer todas as posi√ß√µes
for i in range(len(matriz)):        # Linhas
    for j in range(len(matriz[i])): # Colunas
        print(f"[{i}][{j}] = {matriz[i][j]}")

üìê Diagonal Principal

A diagonal principal de uma matriz quadrada NxN s√£o os elementos onde linha == coluna:

Visualiza√ß√£o:

Matriz 3x3:
[1  2  3]
[4  5  6]
[7  8  9]

Diagonal principal: 1, 5, 9 (posi√ß√µes [0][0], [1][1], [2][2])

Acessar diagonal:

n = 3
matriz = [[1,2,3], [4,5,6], [7,8,9]]

print("Diagonal principal:")
for i in range(n):
    print(matriz[i][i])  # [0][0], [1][1], [2][2]...

üí° Regra:

Elemento est√° na diagonal principal se: i == j
