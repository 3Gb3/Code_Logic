üîµ O que s√£o Sets?

Sets s√£o cole√ß√µes n√£o ordenadas de elementos √∫nicos. S√£o baseados na teoria matem√°tica de conjuntos.

Caracter√≠sticas dos Sets:

  • üö´ Sem duplicatas - cada elemento aparece apenas uma vez
  • üîÑ N√£o ordenados - n√£o mant√™m ordem espec√≠fica
  • ‚úèÔ∏è Mut√°veis - podem ser modificados ap√≥s cria√ß√£o
  • ‚ö° Busca r√°pida - verifica√ß√£o de pertin√™ncia O(1)
  • üßÆ Opera√ß√µes matem√°ticas - uni√£o, interse√ß√£o, diferen√ßa

Cria√ß√£o de Sets:

üßÆ Opera√ß√µes Matem√°ticas

Uni√£o (‚à™):

Interse√ß√£o (‚à©):

Diferen√ßa (‚àí):

üöÄ Casos de Uso Pr√°ticos

1. Remo√ß√£o de Duplicatas:

# Remover duplicatas de lista mantendo tipo
def remover_duplicatas(lista):
    return list(set(lista))

# Preservar ordem
def remover_duplicatas_ordenado(lista):
    visto = set()
    resultado = []
    for item in lista:
        if item not in visto:
            visto.add(item)
            resultado.append(item)
    return resultado

numeros = [1, 2, 2, 3, 1, 4, 3, 5]
print(f"Original: {numeros}")
print(f"Sem duplicatas: {remover_duplicatas(numeros)}")
print(f"Sem duplicatas (ordem): {remover_duplicatas_ordenado(numeros)}")

2. Valida√ß√£o e Filtros:

3. An√°lise de Dados:

# An√°lise de logs
def analisar_ips(logs):
    """Analisa IPs √∫nicos em logs"""
    ips_unicos = set()
    ips_suspeitos = set()
    
    for entrada in logs:
        ip = entrada['ip']
        ips_unicos.add(ip)
        
        # IPs com muitas tentativas
        if entrada['tentativas'] > 10:
            ips_suspeitos.add(ip)
    
    return {
        'total_ips': len(ips_unicos),
        'ips_suspeitos': ips_suspeitos,
        'ips_seguros': ips_unicos - ips_suspeitos
    }

# Exemplo de uso
logs_exemplo = [
    {'ip': '192.168.1.1', 'tentativas': 3},
    {'ip': '10.0.0.1', 'tentativas': 15},
    {'ip': '192.168.1.1', 'tentativas': 5},
    {'ip': '172.16.0.1', 'tentativas': 2}
]

resultado = analisar_ips(logs_exemplo)
print(f"An√°lise: {resultado}")

  • üéØ Sets s√£o ideais para filtros e valida√ß√µes
