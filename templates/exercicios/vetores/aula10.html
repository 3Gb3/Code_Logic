🎯 Objetivo

  • Filtrar dados compostos por múltiplas condições
  • Trabalhar com três listas paralelas
  • Aplicar regra de negócio (aprovação)

📚 Conceitos: Múltiplas Listas Relacionadas

Quando temos vários dados por item, usamos várias listas paralelas:

Estrutura com 3 listas:

nomes = []
nota1 = []
nota2 = []

# Ler dados
for i in range(n):
    nome = input("Nome: ")
    n1 = float(input("Nota 1: "))
    n2 = float(input("Nota 2: "))
    
    nomes.append(nome)
    nota1.append(n1)
    nota2.append(n2)

# Processar dados relacionados
for i in range(len(nomes)):
    media = (nota1[i] + nota2[i]) / 2
    if media >= 6.0:
        print(f"{nomes[i]}: {media:.1f} - APROVADO")

📋 Aplicando Regras de Negócio

Regras de aprovação são comuns em sistemas acadêmicos:

Critério de aprovação:

media = (nota1 + nota2) / 2

if media >= 6.0:
    print("APROVADO")
else:
    print("REPROVADO")

Para este problema:

# Após ler todos os dados
print("Alunos aprovados:")

for i in range(len(nomes)):
    # Calcular média
    media = (nota1[i] + nota2[i]) / 2
    
    # Verificar se aprovado (média >= 6.0)
    if media >= 6.0:
        print(nomes[i])

💡 Ordem de operações:

  1. Ler TODOS os dados primeiro
  2. Depois processar e filtrar
  3. Exibir apenas os aprovados

💡 Problema: Aprovados

Fazer um programa para ler um conjunto de N nomes de alunos, bem como as notas que eles tiraram no 1º e 2º semestres. Cada uma dessas informações deve ser armazenada em um vetor. Depois, imprimir os nomes dos alunos aprovados, considerando aprovados aqueles cuja média das notas seja maior ou igual a 6.0 (seis).
