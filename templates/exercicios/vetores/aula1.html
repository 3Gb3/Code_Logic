📊 O que são Listas?

Listas (ou vetores) são estruturas de dados que permitem armazenar múltiplos valores em uma única variável, organizados em sequência.

💡 Analogia do Dia a Dia

Uma lista é como uma estante de livros:
• Cada posição (prateleira) tem um índice (0, 1, 2...)
• Você pode adicionar, remover ou trocar livros
• Você pode ver quantos livros tem no total

🔧 Criando Listas

Diferentes formas de criar listas:

# Lista vazia
lista_vazia = []

# Lista com números
numeros = [1, 2, 3, 4, 5]

# Lista com strings
frutas = ["maçã", "banana", "laranja"]

# Lista mista
mista = [1, "texto", 3.14, True]

# Lista com range
sequencia = list(range(1, 6))  # [1, 2, 3, 4, 5]

🔍 Acessando Elementos

Usando índices:

frutas = ["maçã", "banana", "laranja", "uva"]

# Acessando elementos (índice começa em 0)
print(frutas[0])    # maçã
print(frutas[1])    # banana
print(frutas[-1])   # uva (último elemento)

# Verificando tamanho
print(len(frutas))  # 4

🔨 Métodos Básicos de Lista

append()

Adiciona elemento no final

lista.append("novo")

insert()

Adiciona elemento em posição específica

lista.insert(1, "meio")

remove()

Remove primeira ocorrência do valor

lista.remove("valor")

pop()

Remove e retorna elemento por índice

elemento = lista.pop(0)

🎮 Exemplo Prático

Lista de Compras:

# Criando lista de compras
compras = ["pão", "leite", "ovos"]

# Adicionando itens
compras.append("queijo")
compras.insert(1, "manteiga")

print("Lista de compras:")
for item in compras:
    print(f"- {item}")

print(f"\nTotal de itens: {len(compras)}")

# Removendo item
compras.remove("ovos")
print(f"Após remover ovos: {compras}")
