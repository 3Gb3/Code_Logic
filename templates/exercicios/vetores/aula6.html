🎯 Objetivo

  • Trabalhar com listas dentro de listas
  • Acessar e modificar elementos em listas aninhadas
  • Percorrer estruturas bidimensionais com loops

📋 Conceito de Listas Aninhadas

Listas aninhadas são listas que contêm outras listas como elementos.

Criação de Listas Aninhadas:

# Criação manual
matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print("Matriz 3x3:")
for linha in matriz:
    print(linha)

⚠️ Cuidado ao Criar com Repetição:

# ❌ ERRADO: Todas as linhas referenciam a mesma lista!
matriz_errada = [[0] * 3] * 3
print(f"Matriz errada: {matriz_errada}")

# Modificar um elemento afeta todas as linhas
matriz_errada[0][0] = 1
print(f"Após modificar [0][0]: {matriz_errada}")  # Todas mudam!

# ✅ CORRETO: Cada linha é uma lista separada
matriz_correta = []
for i in range(3):
    linha = [0] * 3
    matriz_correta.append(linha)

print(f"\nMatriz correta: {matriz_correta}")

# Modificar um elemento afeta apenas uma linha
matriz_correta[0][0] = 1
print(f"Após modificar [0][0]: {matriz_correta}")  # Apenas primeira linha muda

Listas com Tamanhos Diferentes:

# Listas irregulares (cada linha com tamanho diferente)
triangulo = [
    [1],
    [1, 1],
    [1, 2, 1],
    [1, 3, 3, 1]
]

print("Triângulo de Pascal:")
for linha in triangulo:
    print(linha)

🎯 Acessando Elementos

Acesso por Índices:

matriz = [
    [10, 20, 30],
    [40, 50, 60],
    [70, 80, 90]
]

# Acessar linha inteira
primeira_linha = matriz[0]
print(f"Primeira linha: {primeira_linha}")  # [10, 20, 30]

# Acessar elemento específico
elemento = matriz[1][2]  # Linha 1, Coluna 2
print(f"Elemento [1][2]: {elemento}")  # 60

# Acessar com índices negativos
ultimo_da_ultima = matriz[-1][-1]
print(f"Último elemento: {ultimo_da_ultima}")  # 90

Verificação de Índices:

matriz = [
    [1, 2, 3],
    [4, 5, 6]
]

# Acessar com segurança
linha = 1
coluna = 2

if 0 <= linha < len(matriz):
    if 0 <= coluna < len(matriz[linha]):
        elemento = matriz[linha][coluna]
        print(f"Elemento [linha][coluna]: {elemento}")
    else:
        print(f"Coluna {coluna} inválida")
else:
    print(f"Linha {linha} inválida")

✏️ Modificando Elementos

Alterar Elemento Individual:

notas = [
    ["Ana", 7.5, 8.0],
    ["Bruno", 6.5, 7.0],
    ["Carlos", 8.5, 9.0]
]

print("Antes:")
for aluno in notas:
    print(aluno)

# Alterar nota de Bruno
notas[1][1] = 8.5  # Linha 1 (Bruno), coluna 1 (primeira nota)

print("\nDepois:")
for aluno in notas:
    print(aluno)

Modificar Linha Inteira:

tabuleiro = [
    ['X', 'O', 'X'],
    ['O', 'X', 'O'],
    ['X', 'O', 'X']
]

print("Antes:")
for linha in tabuleiro:
    print(linha)

# Substituir linha inteira
tabuleiro[1] = [' ', ' ', ' ']

print("\nDepois:")
for linha in tabuleiro:
    print(linha)

🔄 Percorrendo Listas Aninhadas

Loop com For Each:

matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print("Percorrendo com for each:")
for linha in matriz:
    for elemento in linha:
        print(elemento, end=' ')
    print()  # Nova linha após cada linha da matriz

Loop com Índices:

matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print("Percorrendo com índices:")
for i in range(len(matriz)):
    for j in range(len(matriz[i])):
        valor = matriz[i][j]
        print(f"Posição ({i}, {j}) = {valor}")

Exibir em Formato de Tabela:

tabela = [
    ["Produto", "Preço", "Qtd"],
    ["Arroz", 5.50, 10],
    ["Feijão", 7.00, 5],
    ["Açúcar", 3.50, 8]
]

print("TABELA DE PRODUTOS:")
print("-" * 40)
for linha in tabela:
    for item in linha:
        item_str = str(item)
        espacos = 15 - len(item_str)
        print(item_str + ' ' * espacos, end='')
    print()
print("-" * 40)

➕ Operações com Listas Aninhadas

Somar Todos os Elementos:

numeros = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

soma_total = 0
for linha in numeros:
    for elemento in linha:
        soma_total += elemento

print(f"Soma total: {soma_total}")  # 45

Encontrar Maior e Menor:

valores = [
    [15, 8, 23],
    [4, 16, 42],
    [30, 11, 7]
]

# Encontrar maior
maior = valores[0][0]
pos_maior = [0, 0]

for i in range(len(valores)):
    for j in range(len(valores[i])):
        if valores[i][j] > maior:
            maior = valores[i][j]
            pos_maior = [i, j]

print(f"Maior valor: {maior} na posição {pos_maior}")

# Encontrar menor
menor = valores[0][0]
pos_menor = [0, 0]

for i in range(len(valores)):
    for j in range(len(valores[i])):
        if valores[i][j] < menor:
            menor = valores[i][j]
            pos_menor = [i, j]

print(f"Menor valor: {menor} na posição {pos_menor}")

Contar Ocorrências:

grid = [
    ['X', 'O', 'X'],
    ['O', 'X', 'O'],
    ['X', 'X', 'O']
]

# Contar X e O
count_x = 0
count_o = 0

for linha in grid:
    for simbolo in linha:
        if simbolo == 'X':
            count_x += 1
        elif simbolo == 'O':
            count_o += 1

print(f"X: {count_x} ocorrências")
print(f"O: {count_o} ocorrências")

📊 Operações em Linhas e Colunas

Somar por Linha:

vendas = [
    [100, 150, 200],  # Semana 1
    [120, 180, 210],  # Semana 2
    [90, 160, 190]    # Semana 3
]

print("Vendas por semana:")
for i in range(len(vendas)):
    soma_semana = 0
    for valor in vendas[i]:
        soma_semana += valor
    print(f"Semana {i + 1}: {soma_semana}")

Somar por Coluna:

vendas = [
    [100, 150, 200],  # Semana 1
    [120, 180, 210],  # Semana 2
    [90, 160, 190]    # Semana 3
]

# Número de colunas (produtos)
num_colunas = len(vendas[0])

print("Vendas por produto:")
for coluna in range(num_colunas):
    soma_produto = 0
    for linha in range(len(vendas)):
        soma_produto += vendas[linha][coluna]
    print(f"Produto {coluna + 1}: {soma_produto}")

Calcular Média por Aluno:

notas = [
    ["Ana", 7.5, 8.0, 9.0],
    ["Bruno", 6.5, 7.0, 8.5],
    ["Carlos", 8.5, 9.0, 9.5]
]

print("Médias dos alunos:")
for aluno in notas:
    nome = aluno[0]
    soma_notas = 0
    count_notas = 0
    
    # Somar apenas as notas (a partir do índice 1)
    for i in range(1, len(aluno)):
        soma_notas += aluno[i]
        count_notas += 1
    
    media = soma_notas / count_notas
    print(f"{nome}: {media:.2f}")

💡 Dicas Importantes

  • 📋 Sempre crie cada linha separadamente para evitar referências compartilhadas
  • 🎯 Use len(matriz[i]) para pegar o tamanho de cada linha
  • 🔢 Para acessar elemento: matriz[linha][coluna]
  • 🔄 Use loops aninhados para percorrer toda a estrutura
  • ✔️ Sempre verifique índices antes de acessar para evitar erros
  • 📊 Para somar colunas, percorra todas as linhas com o mesmo índice de coluna
